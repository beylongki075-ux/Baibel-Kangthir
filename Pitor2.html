PITOR KETOK APHI AKIDO

Lam aphrang
Labangso akido ke Pitor thidet pen ningkan thomkep kalite phlikep aphi si ketok. Ahok alam le pudam lote lake Pitor le ketok kali, bonta anut kaprek si Pitor amen pen ketok pusi Baibel kethekser-ser atum kepu. Labangso akido ketok ke 100-125 A.D. aningkan angbong malom lo.

Angdeng kethak
1. Khrist a-ot adur (1:1-11)

2. Jisu Khrist kevangthuji alam (1:12-21)
3. Kaphola lam kethan atum aphan ketam (2:1-22). 4. Kangsaiji aphan Jisu kevangthuji alam (3:1-13).
5. Jisu kevangthuji arni aphan kachesik (3:14-18).

Kardom kachepi :
1.
1*Simon Pitor, Jisu Khrist achakri lapen Ketoitang abang, ilitum a-Arnam lapen kapejok abang ilitum aHemphu Jisu khr1st akanghon pen nelitum asonthot akekroi kelong atum aphan ketok akido.

2*Arnam lapen ilitum aHemphu Jisu aphan kachini pen ajok nanglitum hirjume lapen ning kangsam nang pelong-onang.

Khrist a-ot adur :
3*Jisu Khrist alangli jakong pen ilitum 'kerengji lapen Arnam aphan kamathaji aphan kenangji an epitanglo. Alangli laheihei kepi ke adak atur lapen akeme an ekapelongji aphan ekehang abang Armam aphan ekapachini pen lo.

4*Labangso pen ajok alangli ilitum aphan akethenei lapen ador kesonei alamchak epilo. Labangso alamchak pen ajok ok akar-ithap pen pirthe athak keklang akahi an pen nanglitum chepejok thekpo lapen Arnam achethe apharman longdun thekpo.

5*La apot si nanglitum akekroi aphan ke kam keme pen le pethirnon, kam keme aphan ke kachini ajakong pen le pethirnon.

Kachini ajakong apban ke ning kacheber pen le pethirnon, ning kacheber aphan ke kachechak kachepaningding pen le pethinon, kachechak kachepaning ding aphan ke Arnam aphan kamatha pen le pethirnon. 7*Amam aphan kamatha aphan ke korte kachinghon pen le pethirnon lapen korte kachinghon aphan ke kanghon pen le pethirnon.

8*Nanglitum laheihei a-ot do-o lote ilitum aHemphu Jisu Khrist aphan kachini joikepo lapen ale ta dopo.

9*Laheihei a-ot kave abang aling ke amek ave lapen checharche, lapen alangli papban an ke kak-etlo pu tingnedet.

10*La apot si, korte atum, nanglitum aphan nang kehang lapen nang kang-vai ateng kedo kachepethirji aphan le hungnon. Pima nanglitum la le klem lote nanglitum pap chomathathelo.

11*Lapuson si nanglitum ilitum aphan ekapejok abang ilitum aHemphu Jisu Khrist along kejut un-e arat angbong dodun longpo.

Jisu aphan haki kechodun :
12*Nanglitum kelong ahok alam nanglitum chinilo lapen laso angbong le dothiplo seta laheihei alam neli nanglitum aphan nang kapamathathuji aphan neli ning hang.

13*Neli rengtik an nanglitum aphan nang kapamathathubom ke chok pu si neli kamatha.

14*Pima ilitum aHemphu Jisu Khrist neliphan ne kapeklang ateng neli kethi nangji arni nang tebok-hetlo pu si neli chechinilo.

15*Neli thilot aphi nanglitum laheihei alam kachemathathuji aphan neli avarsai bi tekangpo.

16*Nelitum, nanglitum aphan ilitum aHemphu Jisu Khrist jakong kethe pen kevangthu alam nang kapachini ahut kapahancha atomo le nang kethan kali, nelitum ke alangli hir alangli sang ketheklongvek atum lo.

17*Alangli, Po Arnam along pen aman apeng lapen adak atur kelong ahut, "Labangso abang ke ne kachinghon aSopo lo, alangphan ne ning dok-olo," pu ahir asang lapen adak atur pen ase vang Si haki nang chodunlo.

18*Nelitum ta alangli long halabangso anglong kangthir athak dodun si halabangso sining pen kevang ase arjudunlonglo.

19*Ilitum an sose kachiniji aphan kethanmek alam ta do. Adap ingthangthetik lapen nanglitum aning arlo klirdap vangvetik ke, melur le akangting ajo keturbong ason thakthak nanglitumn labangso alam rim pemenon.

20*Ketok Kangthir along kedo kethanmek alam abangphutin aning kehang ateng peklar longle pu nanglitum chini nangji.

21*Pima ajat akethanmek alam ta monit ajakong pen le kethan kali, bonta monit atum Arnam amen pen kethan ahut Karjong Kangthir ajakong pen si kethan.

Kechokche alam kethan atum :
2.
1*Hako ahut Arnam arat atum angbong kachobei akethanmek atum nang keklang asonthot nanglitunm angbong ta kechokche alam kethan atum nang klangji. Alanglitum rat atum aphan kapepratsei akechokche alam than chonghuji lapen alanglitum aphan kapejok abang aphan ta Hemphu pu hanghedet si amethang namat chepevirdapji.

2*Alanglitum along kapamekphrok akam kacharlidun atum do-oji lapen alnglitum pen ajok ahok atovar aphan ta pinremdetji.

3*Alanglitum amek kethe pen nanglitum aphan kechokche alam nang than si amethang aphan ale chepedoji. Hako pen alanglitum aphan kedin adai nonpu ta do lapen alanglitum vir ta virvekpo.

4*Arnam,pherangke atum pap kachomatha ahut alanglitum aphan ingjinsosepin pen ha norokot var-et si mo kangsai arni nang leletik an kangtingthir aherangpi arlo si pedokoklo.

5*Arnam aphan kamathathe atum aphan kapevirji aphan langpi kapevang ahut Arnam hako ahut amonit atum aphan ingjinsosedet si monit keme abang Noa lapen alangli long anut kaprek bang throksi atum aphan an si pejok chotlo.

6*Sodom lapen Gomora arongsopi heihei me pecho-et si atheklo pacheplang-et mamat pangtheidang si Arnam aphan kamathathe atum kephoji angthek peklanglo.

7*Alangli, kapamekphrok atum akam pen apot aning ke-oi abang, ahok amonit Lot aphan ke pejokphlotlo.

8*Halabangso ahok amonit alanglitum along kedo pen arnivang ketheklong lapen karjulong alanglitum kapamekphrok akam alamjir pen ajok alangli therakduklo.

9*La apot si Hemphu ke Arnam aphan kamatha atum aphan kesung kepho pen pejok thek lapen kangsai arni nang leletik an kahingno amonit atum aphan pesung thek.

10*Jasemet ke a-ok akar-ithap kachepehaibom lapen jakong horang kedo atum aphan kelangret atum aphan pesung thek.

Kahingno atum aphan kapesung :
Lapuson amonit atum ke chepehaidong si phererepin pen adak atur kelong atm aphan pinrem hai binong.

11*Bonta alanglitum apar kangtang kangsak lapen jakong horang kedo apherangke atum ke Hemphu angno alanglitum aphan pinremre.

12*Halaheihei amonit atum ke, amut kave, angthin kave amarat api ason thakthak lo, halatum ke bang kapachonepji aphan an si kedo chot, halatum ke kachininepin alam si kapinrem.

13*Halatum ke halatum kahingno keklem apot kesung pho si marat api ason vir-etji. Halatum ke losini sini ta kecho kejun arong chepi chot. Halatum ke nanglitum along nang kecho nang kejundun ahut ta a-ot aji kave pen ajok nanglitum angbong therak meme lapen man peng ta ave-idet.

14*Halatum ke beisa akam lok si kamatha lapen pap kachomatha ta dor un-e.

Halatum kekroi kethirthe atum aphan chomosoibomlo. Halatum amek kethe lok si aning kedo. Halatum ke hirjuno kelongtang atum lo.

15*Halatum ahok atovar ke damdedet si avive si damrong lapen kahingno keklem pen ale kachepelong abang Beor asopo Balam ason si kacharlidun.

16*Halabangso abang akam kahingno apot tam longlo, pima alam kave agadha monit ason ningjeser si halabangso akethanmek abang kapangcham khangcheklo. akam

17*Halatum ke alang kave alangmek lapen tomon kepon ahunmelan ason thakthak lo; akangtingthir aherangpi ke halatum aphan si kapanongpi lo.

18*Halatum kelu alam kethan kachepindak pen lapen kapamekphrok akam heihei pen kaphola alam kekroi atumn pen kejokchit atum aphan chomosoilo.

19*Halatum aphan kapejokji alam halatum chechak, bonta halatum mamat ta kahingno keklem atum pinak, pima monit atum ke halatum aphan kehai un ason aling aban lo.

20*Halatum ilitum aphan ekapejok abang ilitum aHemphu Jisu Khrist aphan kachini pen pirthe akahingno pen jokdet aphi puthot halaheihei akahingno along chingnang dunthulok ra kahingno pehai longdet lote, halatum aphrang apar si hingno munakpo.

Kerip alam :
